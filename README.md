## 📖 About

Miners4k: Unearthed is a fan-made modern remake of the original Java 4K entry. You command a swarm of autonomous miners who explore the underground, collect gold, and bring it back to the surface. Help them by digging tunnels, building ramps, and guiding them to riches before time runs out.

---

## 🕹️ Controls

### Desktop
| Input | Action |
|---|---|
| `Arrow Keys` / `WASD` | Move Camera |
| `Mouse Wheel` | Zoom In / Out |
| `Left Click + Drag` | Dig Dirt |
| `Right Click + Drag` | Build Dirt |

### Mobile
| Input | Action |
|---|---|
| **1 Finger Drag** | Dig / Build (use ⛏️ 🧱 buttons) |
| **2 Finger Drag** | Pan Camera |
| **Pinch** | Zoom In / Out |

## ✨ Features

- 🌍 **Infinite-looking terrain** — the world extends visually beyond the map edges
- 💡 **Dynamic miner headlamps** — miners cast real-time directional light
- 🤖 **Smart miner AI** — miners follow gold trails and alert companions nearby
- 💨 **Particle system** — sparks, smoke, and gold shimmer
- 🎚️ **Speed controls** — 1x to 5x game speed
- 📱 **Mobile-first UI** — multitouch pan, pinch-to-zoom, compact HUD
- 🔊 **Sound effects** — digging, gold collection, explosions
- 🏆 **7 progressive levels** — each larger and more challenging
- ☠️ **Slime hazards** — explosive traps that blow up unwary miners
- 🪨 **Rock formations** — obstacles requiring strategy

---

## 🛠️ Tech Stack

- **TypeScript** + **Vite**
- **HTML5 Canvas** (pixel-level rendering at 320×240, scaled up)
- **Vanilla CSS** with Glassmorphism HUD
- **PWA-ready** (installable on mobile)

## 🚀 Running Locally

```bash
npm install
npm run dev
```

Build for production:
```bash
npm run build:zip   # generates miners4k-web.zip ready for itch.io
```
